This document is aimed to guide in the process of creating a Linked Data (LD) version of a lexical resource, particularly a bilingual dictionary. These guidelines contain advice on the vocabularies selection, RDF generation process, and publication of the results. As result, the converted language resource is more interoperable and easily accessible on the Web of Data by means of standard Semantic Web technologies.

This document was published by the Best Practices for Multilingual Linked Open Data comunity group. It is not a W3C Standard nor is it on the W3C Standards Track.

There are a number of ways that one may participate in the development of this report:

  • Mailing list: public-bpmlod@w3.org
  • Wiki: Main page
  • More information about meetings of the BPMLOD group can be obtained here
  • Source code for this document can be found on Github.
